Authentication with Amazon Cognito

A Cognito User Pool issues JWT tokens. IAM roles scoped per user restrict S3 access to personal prefixes like s3://bucket/user-id/.

Secure File Upload

Lambda generates pre-signed URLs so the browser uploads directly to S3 without exposing credentials.

Lessons Learned

  • Always use pre-signed URLs; never expose long-lived keys.
  • Keep S3 buckets private; enforce least-privilege IAM policies.
  • Enable S3 versioning & lifecycle rules for archival tiers.
